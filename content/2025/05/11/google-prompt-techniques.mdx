## Key Techniques Toolbox

<ol className="list-decimal ml-6 space-y-1">
  <li>**Zero‑shot / General** — simple instruction.</li>
  <li>**One‑shot / Few‑shot** — add examples to teach style.</li>
  <li>**System • Role • Context** — set persona / task / background.</li>
  <li>**Step‑Back (Two‑Step)** — ask a broad “why/what” first, feed answer into real task.</li>
  <li>**Chain of Thought** — tell model to “think step‑by‑step” for reasoning.</li>
  <li>**Self‑Consistency** — sample multiple CoT runs, vote on answer.</li>
  <li>**Tree of Thoughts** — explore parallel reasoning branches for hard problems.</li>
  <li>**ReAct (Reason + Act)** — interleave analysis with tool calls / searches.</li>
  <li>**APE** — automatic prompt engineering; let the model draft & rank prompts.</li>
  <li>**Code prompts** — ask to write, explain, translate, debug code.</li>
  <li>**Multimodal** — mix images/audio/text when the model supports it.</li>
</ol>
